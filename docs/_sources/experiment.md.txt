# experiment

- 当多重性为 0 时，输出高电平
- 当多重性大于等于 1 时，默认输出低电平，有触发时为高电平

## 在线监视

修改好参数配置文件 settings.ini

执行
```bash
progfippi
```
写入设置参数

**需要注意的是，当获取采集时候，不允许执行该程序**

在网页中即可查看参数设置情况，scale计数情况
